> **Updated concept:**
> OTU should be understood not only as a deep-ocean aeration device, but also as a **bidirectional spiral ocean vertical-flow tuning unit**.

The mode overview below summarizes how these functions are separated to avoid conflating aeration, cooling, diffusion, plankton support, energy recovery, and ecological safety.

## OTU Mode Overview

| Mode | Direction / Role | Main Function | Notes |
|---|---|---|---|

| Reverse Cooling Mode | Upward / Surface | Cold deep-water lifting and direct sea-surface / upper-layer cooling | Operated only under high-temperature or necessary conditions |
| Diffusion Mode | Wide-angle / Spiral | Dilution, mixing, and local concentration avoidance | Supports safe release of cold deep water and nutrients |

# Abstract

Current climate mitigation strategies primarily focus on reducing atmospheric CO₂ concentrations.

This paper proposes an alternative framework:

# Direct Planetary Cooling (DPC)

through the deployment of:

# Ocean Tuning Units (OTU)

By implementing autonomous oceanic regulation infrastructure utilizing:

* inverse-rotational helical propulsion,
* buoyancy-driven vertical convection,
* microbial restoration,
* and distributed thermal regulation,

the OTU framework proposes a low-energy method for bypassing deep-sea hydrostatic pressure barriers, re-establishing planetary vertical circulation, and restoring Earth's thermal equilibrium.

The system is designed to function as:

> a planetary-scale “Heart-Lung Bypass”

for restoring oceanic metabolic circulation before projected 2030 climatic tipping points become irreversible.
